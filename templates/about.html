Following a course

You can see all courses you follow on yout dashboard

When following a course, you have access to all its classes and receive notifications for new events

When watching a Youtube video, you can write annotations on it to watch later(annotations are synchronized with video time)

You can download class documents

For any question related to class, you have a 'comments' area on every class. Or even just to discuss about it

ou can also send a message to the person who teach the class through his profile page

Creating a course

You can create a new course through the 'My Courses' link on the left nav bar

Each course can contain as much classes as you need

Each classe will contain videos from Youtube, Uploaded documents, Google Drive documents and additional resources as hyperlinks

You can also start a live class thourgh Hangout OnAir, the recorded class will be available as a class video after hangout end

Classes has a special 'comments' area for students make questions or discuss about it

You can add an calendar event associated to a course, like: 'New Live class'

APIs used

Google Appengine(TaskQueue/Deferred, Email, Images, Blobstore, Memcache, Search API, URL Fetch)

Youtube API

Google Calendar API

Hangout API

Google Drive API
